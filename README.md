# Лабораторная работа 1: Лексический анализ псевдокода

## Описание проекта
Система для лексического анализа учебного языка псевдокода. Реализован полнофункциональный лексический анализатор, способный распознавать ключевые слова, операторы, литералы и структуры управления.

## Структура проекта
009/
├── docs/ # Документация
├── src/ # Исходный код
│ ├── grammars/ # Грамматики ANTLR
│ └── *.py # Основные модули
├── tests/ # Тесты и тестовые случаи
├── examples/ # Примеры программ
└── config/ # Конфигурационные файлы

text

## Быстрый старт
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск основного лексера
python src/lexer.py

# Запуск тестов
python tests/run_tests.py

# Проверка структуры
python check_structure.py
Возможности
✅ Распознавание ключевых слов (if, while, for, etc.)

✅ Поддержка арифметических операторов (+, -, *, /)

✅ Операторы сравнения (==, !=, <, >, <=, >=)

✅ Логические операторы (&&, ||, !)

✅ Обработка чисел, строк и идентификаторов

✅ Поддержка комментариев (#)

✅ Позиционирование токенов (строка, столбец)
